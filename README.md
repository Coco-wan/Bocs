# Bocs

> 一个小巧的阅读管理工具

[线上Demo](http://bocs.leanapp.cn)

## 运行

```bash
git clone git@github.com:yrq110/Bocs.git

cd bocs

npm install 

npm run dev
```

## 首页

![](https://github.com/yrq110/Bocs/blob/master/screenshots/home.png)

## 技术与库

* vue + vuex + vue-router
* less + webpack
* axios (网络请求)
* lockr (本地存储)
* vue-lazyload (图片懒加载)

## 特性
* 组件内均通过actions触发mutation修改state
* 使用flex与rem实现响应式界面
