# Bocs

> 一个小巧的阅读管理工具

[线上Demo](http://bocs.leanapp.cn)

## 运行

```bash
git clone git@github.com:yrq110/Bocs.git

cd bocs

npm install 

npm run dev
```

## 预览

## 所用技术与库

* vue + vuex + vue-router (vue全家桶)
* axios (网络请求)
* lockr (本地存储)
* vue-lazyload (图片懒加载)
* webpack
* less
所有操作通过action方式提交

## 其他
* 所有数据的CURD操作通过actions触发
* 使用flex与rem实现响应式布局
